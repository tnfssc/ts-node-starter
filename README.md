# [ts-node-starter](.)

## Usage

```sh
npm install # install dependencies
npm run build # build the project
npm run start # start the project (after building)
npm run dev # start the project in development mode (restarts server on changes)
npm run test # run tests (watch mode)
npm run coverage # run tests with coverage (no watch mode)
npm run lint # run linter
npm run check # run type checker
npm run format # run formatter (prettier)
```
